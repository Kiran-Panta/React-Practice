Defense Question

CSS Grid ra Flexbox ko difference ke ho? Flexbox is for 1D layouts (either row or column), Grid is for 2D layouts (both rows and columns together).

Kuna use garne kuna situation ma? Use Flexbox for simple alignments like navbars or card rows. Use Grid for complex layouts like dashboards or magazines.
