## Challenges Faced

### Bootstrap JS Dependencies
Understanding how Bootstrap components require JavaScript was tricky. The data-bs-toggle and data-bs-target attributes do nothing without the Bootstrap JS library. This taught me about progressive enhancement and JavaScript dependencies.

### Modal Accessibility
Implementing proper focus management in modals was complex. I learned about focus trapping, keyboard navigation, and preventing body scroll. The React Portal for rendering outside DOM tree was also new.

### Form Validation UX
Deciding between different feedback methods (toast vs inline errors) required UX thinking. I learned that immediate inline feedback is best for forms, while toasts work well for success confirmations.
