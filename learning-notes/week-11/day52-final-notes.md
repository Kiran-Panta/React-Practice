## ❌ Things I Must Not Forget (10 items)

1. **useEffect dependency arrays** — empty [] runs once, missing deps causes bugs
2. **React keys in lists** — unique keys for proper reconciliation
3. **Event handler binding** — arrow functions or bind() in class components
4. **API error handling** — always catch and display errors to users
5. **Form validation** — client-side validation doesn't replace server validation
6. **CSS specificity** — ID > class > element selectors
7. **JavaScript closures** — functions capture variables from outer scope
8. **React Router params** — useParams() returns string, convert to numbers if needed
9. **Local storage** — JSON.stringify() and JSON.parse() for objects
10. **Git commit messages** — clear, descriptive messages in imperative mood
