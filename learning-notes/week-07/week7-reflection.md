# Week 7 Reflection — Kiran Panta

## What I Learned This Week

This week was all about polishing the app and making it look professional! I learned so much about Tailwind CSS and how to customize it.

### Key Concepts:
- **Tailwind Play CDN**: Standalone practice without build tools
- **Custom Tailwind Theme**: Adding brand colors, fonts, and spacing
- **Responsive Design**: sm:, md:, lg: breakpoints and mobile-first approach
- **Dark Mode**: class strategy vs media strategy
- **React Portals**: Rendering components outside normal DOM tree
- **Custom Hooks**: useToast for managing toast notifications
- **Intersection Observer**: Animating elements when they come into view

### UI Components:
Built reusable components like Toast, Modal, ScrollToTop, and LoadingSpinner. Learned about component composition and prop drilling.

### Tailwind Deep Dive:
- Utility-first philosophy vs writing custom CSS
- Custom theme extension with tailwind.config.js
- When to use @apply for component classes
- Responsive prefixes and their pixel values

## Challenges Faced
- Understanding React Portals took some time - the "teleportation" concept
- Customizing Tailwind theme properly
- Making smooth animations with CSS transitions
- Managing toast state with auto-dismiss

## What I Enjoyed
- Seeing the app transform from basic to professional-looking
- Building the dark mode toggle - it felt like a real feature
- Creating the animated skill bars that fill up on scroll
- The satisfaction of a polished, complete portfolio

## Goals for Next Week
- Learn more advanced React patterns
- Maybe add some backend integration
- Continue improving the UI/UX

The app now looks like a real professional website! I'm proud of how far I've come from basic HTML/CSS to a full React application with routing, API calls, and beautiful styling.
